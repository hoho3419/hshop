# hShop 홈 쇼핑 배달 어플리케이션📌
## STACK USED
* ReactJS (Typescript) <br>
* Recoil <br>
* TailwindCSS <br>
* Firease <br>
## Production Start Date📌
* 2023년 6월 8일 ~
## Function list📌
* 회원가입
* 로그인,로그아웃
* 구글 로그인
* 장바구니(추가,수정,삭제)
* 프로필 변경 (유저 데이터 변경)
* Menu Filter 기능
* Google Map API
